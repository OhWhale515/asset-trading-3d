# 3D Asset Trading View

> Market data projected onto an interactive 3D terrain for visual analysis.

Day 3 of the 25days25 sprint. Tier 3 (Viral). Group A.

---

## Quick start

```bash
npm install
npm run dev
```

The app runs at `http://localhost:3000`.
